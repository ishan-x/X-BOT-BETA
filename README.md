## Deployment

This project is configured for automatic deployment to GitHub Pages. Simply push to the `master` or `main` branch, and the GitHub Actions workflow will automatically build and deploy the site.

### Manual Deployment

1. Build the project:
   ```bash
   npm run build
   ```

2. The built files will be in the `dist/` directory

3. Deploy the `dist/` folder to your hosting service
